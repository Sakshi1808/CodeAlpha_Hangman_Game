🎮 Hangman Game (Python)

1.Description

This is a simple text-based Hangman game built using Python.
The player tries to guess a randomly selected word one letter at a time within a limited number of attempts.

---

2.Features

- Random word selection from a predefined list
- Maximum of 6 incorrect guesses
- Displays correctly guessed letters and blanks
- Shows remaining guesses
- Prevents repeated guesses
- Play again option after each game

---

3. Technologies Used

- Python (Beginner Level)
- Concepts:
  - Random module
  - While loop
  - If-else conditions
  - Strings and Lists

---

4. How to Run the Game

i. Make sure Python is installed on your system

ii. Download or clone this repository

iii. Open terminal or command prompt

iv. Run the program:
   
   python hangman.py

---

5.How to Play

- The game will display a word with missing letters ("_ _ _ _")
- Enter one letter at a time
- If correct → letter is revealed
- If wrong → you lose one attempt
- You have 6 incorrect guesses
- Guess the word before attempts run out to win

---

6. Example Gameplay

Word: _ _ _ _ _
Remaining guesses: 6
Enter a letter: a
Correct!

Word: a _ _ _ _
